mbc-common
==========

Common code for mbc-playout and mbc-mosto

REQUIREMENTS
============

* Redis (`$ apt-get install redis-server )

Versioning
----------

major.minor.patch

* major: backwards-incompatible changes
* minor: new features
* patch: bugfix
